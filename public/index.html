Sweater Weather

Sweater Weather is a Rails API that aggregates all forecast/weather, background image, book, and road trip data that a front end application could need.

End Points

GET /api/v1/forecast?location=denver,co

GET /api/v1/backgrounds?location=atlanta,ga

GET /api/v1/book-search?location=albuquerque,nm&quantity=2

GET /api/v1/backgrounds?location=atlanta,ga
